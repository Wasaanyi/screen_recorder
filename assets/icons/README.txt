Place your application icon here.

Required format: PNG
Recommended sizes:
- icon.png (256x256 or 512x512)

For Windows builds, you may also want to create an .ico file:
- icon.ico (contains multiple sizes: 16x16, 32x32, 48x48, 256x256)

The icon will be used for:
- Application window icon
- System tray icon
- Installer icon
- Desktop shortcut icon

You can create icons using tools like:
- GIMP (free)
- Photoshop
- Online icon generators
- Figma
